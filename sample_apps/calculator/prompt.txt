Build a calculator app with a display at the top showing the current value, and a grid of buttons below for digits 0-9, operators (+, -, *, /), a clear button (C), and an equals button (=). Use a Card for the display area with a large Text component showing the current value bound to /display. Arrange the buttons in Rows of 4 using a Column layout. Each button should send a serverAction with the button's value. Use primary style for the equals button and secondary for operators.
