Build a calculator app with a display at the top showing the current value, and a grid of buttons below for digits 0-9, operators (+, -, *, /), a clear button (C), and an equals button (=). Use a large Text component showing the current value bound to /display. Arrange the buttons in Rows of 4 using a Column layout. Each button should use a functionCall action with updateDataModel to modify the data model directly — use if/concat/calc/toNumber/toString functions for digit appending, operator evaluation, and display formatting. Style with dark background (#1C1C1E), round buttons (cornerRadius: 33), and orange (#FF9F0A) for operators/equals.
