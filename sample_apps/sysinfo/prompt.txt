Build a system info dashboard that loads real native libraries via FFI and displays their version information. Use loadLibrary to load libcurl.dylib, libsqlite3.dylib, and libz.dylib from the system. Call curl_version() (returns string, no args), sqlite3_libversion() (returns string, no args), and zlibVersion() (returns string, no args) using functionCall in component props. Also call compressBound (uint64 → uint64) with a sample value to show a computed result. Display each library as a Card with the library name as title and the version string as body text. Use a dark theme (#1C1C1E background) with blue accent (#0A84FF) for headings. Include a "Compression" card showing compressBound(10000) as a practical FFI demo. Layout: Column of Cards with gap 12, padding 16.
