# Getting Started

Canopy turns text descriptions into native macOS applications. No webview, no Electron --- real AppKit widgets driven by declarative JSONL.

## Prerequisites

- **macOS 13+** (Ventura or later)
- **Go 1.25+** (for building from source)
- An API key for at least one LLM provider (if using prompt mode)

## What You Will Learn

1. **[Installation](installation/)** --- Clone the repo, build the binary, set up API keys
2. **[Your First App](first-app/)** --- Run a JSONL file, write a hello world, generate from a prompt
3. **[CLI Reference](cli-reference/)** --- Every mode, flag, and make target available to you

Once you are comfortable with the basics, explore the [Components](/canopy/components/) reference to see every widget Canopy supports, or read the [Building Apps](/canopy/guides/building-apps/) guide for data binding, layouts, and callbacks.
